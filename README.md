#  My Portfolio

A modern, responsive personal portfolio showcasing my projects, skills, and experience. Designed with clean UI, smooth animations, and a focus on performance, this portfolio highlights my work while offering an intuitive and engaging user experience.

---

##  Features

-  Modern and visually appealing design  
-  Fully responsive across all devices  
-  Project showcase with detailed descriptions  
-  About section with background & expertise  
-  Contact form for easy communication  
-  Fast loading & smooth interactions  
-  Clean UI with thoughtful typography and spacing  

---

##  Tech Stack

- **Next.js / React**  
- **Tailwind CSS / Shadcn UI**  
- **Framer Motion**  
- **Vercel (Deployment)**  

---

##  Installation & Setup

Clone the repository:

```bash
git clone https://github.com/Samar-Srivastava/My-Portfolio.git
cd My-Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Your portfolio will be available at:

```arduino
http://localhost:3000
```
